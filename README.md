# 🎮 Simple Doodle Jump–Like Platformer

A very simple **Doodle Jump–inspired platformer** built with Python and Pygame.<br>
Jump on platforms and try not to fall!

---

## 🚀 Features

* 🕹️ Basic player movement
* 🪜 Simple platforms
* 💀 Game over when you fall
* 🔄 Restart with **SPACE**

---

## 🎮 Controls

| Key   | Action            |
| ----- | ----------------- |
| ← / → | Move left / right |
| SPACE | Restart           |

---

## 🧠 Gameplay

Move left and right to land on platforms.<br>
Keep jumping as long as possible.<br>
If you fall → **Game Over**
